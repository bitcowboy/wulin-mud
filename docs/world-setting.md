# 清河镇世界观 · World Setting

> 这份文档定义 v0.1 验证场景的世界设定。是设计师参考、LLM grounding、和玩家手册的共同底本。

## 一、时代背景

**架空唐风**——不锚定真实历史年份，借用唐代意象作为氛围底子：

- 服饰、建筑、礼仪、官制大致取唐风
- 武学体系、江湖派别属于架空
- 不出现真实历史人物
- 也不必严守史实细节（例如某些器物的考古准确性）

这个设定的好处是：玩家熟悉的视觉/文化符号 + 充分的虚构自由。

## 二、清河镇

### 地理

清河镇位于一条名为"清河"的水道与一条官道的交汇处。距京畿约二十日水程，距西域商道入口约十二日陆程。是商旅东西往来的一个常见歇脚点。

镇子规模约二百户人口，主街一条贯穿东西，从东头码头到西头镇门。常住居民约八九百人，加上往来过客，日常在镇者千人上下。

镇内主要建筑（玩家可去的地点）：

| 地点 | 位置 | 主要 NPC |
|---|---|---|
| **清河码头** | 镇东头 | 阿青（跑堂会来接客） |
| **清河客栈** | 主街东段 | 柳娘子、阿青 |
| **回春堂** | 主街中段偏南 | 孙婆婆 |
| **王家茶肆** | 主街中段 | 王老九 |
| **福顺镖局** | 主街中段偏北 | 赵掌柜、二郎 |
| **老铁的铁炉铺** | 镇西巷子 | 老铁、铁莲 |
| **何家当铺** | 主街西段 | 何当家 |
| **方先生家** | 巷子深处 | 方先生 |
| **镇衙** | 主街最西 | 钱捕头 |
| **赌坊（不挂招牌）** | 镇北边角 | 胡三 |
| **镇尾破庙** | 镇西外 | 疯婆子 |
| **沈先生的临时落脚处** | 客栈内（常住客房） | 沈先生 |

### 经济

- 镇上经济以服务过路商旅、本地农产、药材集散为主
- 本地特产：清河水产、几味本地草药
- 不产矿产、不产丝绸——所以铁器和好布料都靠外面运
- 货币：铜钱为主，银两少见

### 治安

- 名义上归本州州衙管辖，实际由镇衙的钱捕头管事
- 钱捕头懒散，"小事不管，大事不出"
- 没有大规模驻军，江湖人来了基本靠自觉
- 福顺镖局对官府"自愿性协助"，实际是地方上半官半民的秩序维护者

## 三、三股暗中势力

这是整个验证场景的戏剧底盘。v0.1 不需要玩家显式发现这些势力，但所有 NPC 的行为都受其影响。

### 3.1 朝廷暗探系统

**核心人物**：柳娘子（清河客栈老板娘）

**目的**：监视过往江湖人、收集情报、必要时直接行动。

**特征**：
- 表面是个守寡的客栈老板娘，30 岁，精明客气
- 实际是某个朝廷情报机构的本地节点
- 阿青是她培养的"小耳目"
- 她对所有住店客人做基础侧写，每月通过某个渠道上报
- 她不挑事，但会借力打力

### 3.2 正派背景（剑派暗桩）

**核心人物**：赵掌柜（福顺镖局掌柜）

**目的**：监视西线商道，为某剑派提供消息、必要时收容受伤同门。

**特征**：
- 表面是个 45 岁的镖局掌柜，豪爽仗义
- 实际是某剑派退隐弟子，奉命留守
- 镖局的几个镖师，有的是真镖师，有的是剑派外围
- 与朝廷不冲突也不合作，"井水不犯河水"
- 对正派来人客气接待，对邪派来人警惕但不主动出手

### 3.3 邪派联络点

**核心人物**：沈先生（外地药材商）

**目的**：用药材运输线传递情报和小宗违禁品。

**特征**：
- 表面是 40 岁的外地药材商，温文尔雅
- 实际是某教派的联络点
- 每年定期来清河镇 4-5 次，每次住一两周
- 用回春堂等药铺作为掩护交易
- 他出手大方，对孙婆婆这种人特别客气，但从不深交
- 他求人办的"小忙"会越来越越界

### 3.4 三方关系

- 三方互不知道对方在镇上有节点
- 但都隐约怀疑"这镇上不简单"
- 玩家是变量——一个外来年轻侠客的行为可能打破某个平衡

**v0.1 不要求玩家发现这些**。v0.2 中如果玩家行为模式正确，可能逐步揭开。

## 四、玩家角色

玩家扮演一个 19-25 岁的年轻侠客，性别可选。

**设定**：
- 出身不详（玩家可以自己脑补）
- 武功路数：浅薄但能打——能打跑泼皮，打不过真正的高手
- 钱袋：初始 50 文铜钱（够吃几顿饭）
- 装备：一把铁剑（不锋利但能用）、一身洗得发白的青衫
- 行李：一个布包，几件换洗衣服

**开场情境**：玩家随一艘客船到达清河镇码头，下船时船家说"年轻人在这镇上歇歇脚，前面路还远着呢"。

之后的一切由玩家自己探索。

## 五、世界规则

### 时间

- 1 真实分钟 = 10 游戏内分钟（可配置）
- 一个游戏内"日"是 24 游戏内小时
- 大部分 NPC 有作息：早起开店、午后休息、傍晚关门、夜里在家
- 玩家不会困也不会饿（v0.1 不做生存系统）

### 空间

- 玩家通过 `去 <地点>` 移动
- 相邻地点之间需要 5-15 游戏内分钟
- 一日内可以走遍全镇
- 镇外暂时不开放（v0.1 限定在镇内）

### 战斗

- v0.1 做最简化的回合制战斗：双方互攻，每方有 hp，归零则败
- 玩家不会真的"死"——失败会被路人救回客栈，但会产生 Memory 和后果
- 杀死 NPC 是允许的——但每个 NPC 都是 unique，杀了就没了，且会引发连锁反应（治安、关系、传闻）

### 时间流逝

- 玩家不操作时，世界继续 tick
- 离线再上线，世界已经往前走了（按真实时间换算）
- NPC 不会无限等玩家——她们会去做自己的事

## 六、命名与文风约定

### 命名

- 人物：用"姓+尊称/职业"（孙婆婆、王老九、赵掌柜、柳娘子）
- 地点：朴实直白（清河码头、王家茶肆、福顺镖局）
- 物品：唐代或江湖风格（止血膏、跌打方、宽口铁剑）
- 不用现代外来词（如"咖啡店"、"打卡")

### 文风

- 旁白：简净、不堆形容词、留白
- NPC 对话：符合该 NPC 的 speech_style
- 系统提示：清晰、不戏剧化、不滥用"！"

参考样本：
- 古龙：留白、节奏、句子短
- 金庸（中后期）：场景描写朴实
- 极乐迪斯科：内心独白可以有，但克制

避免：
- 网文式的大段心理描写
- AI 翻译腔（"作为一位侠客，您应该..."）
- 现代汉语的拗口表达

## 七、扩展边界

**v0.1 不做**：
- 镇外世界
- 主线剧情
- 帮派系统的玩家可见层（玩家可以怀疑，但系统不显式给出）
- 玩家自定义出身

**v0.2 计划**：
- 玩家可以揭开三股势力之一
- 镖局事件作为第一条因果传导链验证
- 玩家行为开始影响镇上整体氛围（治安、繁荣度）

**v1.0 愿景**：
- 镇外开放
- 多个小镇连成"江湖"
- 多人在同一服务器，玩家行为产生服务器历史
